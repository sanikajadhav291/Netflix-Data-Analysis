Netflix Data Analysis (Python + Tableau) Project

Tools Used:
- Python (Pandas, Matplotlib)
- Tableau

Project Description:
This project involves cleaning and analyzing Netflix dataset to find useful insights.

Work Done:
- Cleaned missing values and removed duplicates
- Converted date columns for analysis
- Performed exploratory data analysis
- Created visualizations using Python
- Built interactive dashboard in Tableau

Key Insights:
- Netflix has more Movies than TV Shows
- Content increased till 2019
- USA and India produce most content
- Most content is rated for adult audience

Conclusion:
This project helps understand Netflix content trends and user targeting.
